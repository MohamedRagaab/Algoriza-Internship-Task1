# 7Krave app
It is a food delivery app developed with Flutter and Dart

## Languages and Packages 📑
* Flutter 
* Dart
* Country Picker

## IDEs 💻
* VS code
* Android Studio

## Platforms 📱
* android 
* ios

## Setup
* clone the repository and run the app on any plugged ios or android Emulator or physical device.
